## Mastering Your Locked 2 Private Server Commands

The most significant change in the transition to the 2026 version of the game is the removal of legacy chat commands. Previously, players had to memorize strings like `/CMD/Scoreboard`. Now, all administrative actions are centralized within the "M" menu. Only the private server owner or designated admins can access these tools, providing a much cleaner experience for the rest of the players.

Within the command interface, you have granular control over the server state, player behavior, and match flow. The system is divided into three primary categories: Server, Player, and Match Settings.

### Essential Server & Player Commands

| Command Category | Feature | Description |
| :--- | :--- | :--- |
| **Server** | **Server Lock** | Prevents new players from joining the session mid-match. |
| **Server** | **Ball Spawning** | Toggle the ability to spawn balls using the **B** key. |
| **Server** | **Reset Flow** | Instantly resets the flow bars for every player on the field. |
| **Player** | **Referee Mode** | Assigns a player to oversee the match without active play. |
| **Player** | **Freeze/Unfreeze** | Temporarily restricts a player's movement to stop griefing. |
| **Player** | **Blacklist** | Permanently bans a user from your specific private server. |

> **💡 Tip:** Use the "Reset Balls" command frequently during training to clear clutter from the field and maintain high server FPS.

## Multipliers and Match Customization

Managing a **locked 2 private server** effectively requires a deep understanding of multipliers. These settings allow you to deviate from standard physics to create unique training scenarios. For example, the "Tester World Cup" often uses specific multipliers to balance the gameplay for 11v11 formats.

The ball gravity setting is particularly sensitive. The default value is **7**. Setting this to a lower number, such as 1, will cause the ball to float almost indefinitely, while higher numbers like 20 will make the ball drop like lead.

### Gameplay Multiplier Breakdown

| Multiplier Name | Default Value | Effect on Gameplay |
| :--- | :--- | :--- |
| **Ball Gravity** | **7** | Controls how fast the ball falls; crucial for aerial training. |
| **Power Multiplier** | 1.0 | Increases or decreases the velocity of all kicks. |
| **Flow Charge** | 1.0 | Adjusts how quickly players earn their Flow state. |
| **Hitbox Multiplier** | 1.0 | Modifies the detection radius for ball contact. |
| **Tackle Cooldown** | 1.0 | Changes the frequency at which players can slide tackle. |

In addition to physics, you can now toggle between different field sizes. The **Large Field** is designed for traditional 11v11 matches, while the **Small Field** is significantly more appropriate for 5v5 or 7v7 "autos" where the action is faster and more condensed.

## Character Heights and Flow Meta

Height in this game is no longer just a cosmetic choice; it is one of the top two factors influencing your playstyle. Unlike the previous version, the 2026 update introduces specific percentage-based buffs and debuffs for different heights. Choosing the right height for your position is vital when playing within a **locked 2 private server** competitive environment.

*   **5'3" - 5'7" (Small):** Best for high mobility and quick turning. These players often act as wingers or agile playmakers.
*   **5'9" (Mid-Range):** The "All-Rounder." This height offers the lowest skill cooldowns in the game, making it the most versatile for midfielders.
*   **6'0" - 6'3" (Tall):** Essential for Strikers and Center Backs. The increased reach allows for better headers and defensive interceptions.

### Height Performance Comparison

| Height | Primary Buff | Best Position | Cooldown Modifier |
| :--- | :--- | :--- | :--- |
| **5'3"** | Max Speed | Winger | Standard |
| **5'9"** | **Cooldown Reduction** | Midfielder | **Fastest** |
| **6'0"** | Balanced Power | Striker | Standard |
| **6'3"** | Reach/Strength | Center Back | Slowest |

When it comes to Flow types, the community is currently split between **Off Ball** and **Burst**. Off Ball is widely considered superior for competitive 11v11 because it provides a consistent speed buff that drains significantly slower when you aren't holding the ball. Burst is better suited for 4v4 matches where you are expected to make solo plays frequently.

## Advanced Techniques: Bicycle Kicks and Weapon Stacking

One of the most exciting additions to the 2026 mechanics is the **Manual Bicycle Kick**. Unlike before, where you needed a specific weapon to perform this move, it can now be done with any build. To execute it, hold your backwards key (typically **S**), charge your kick, jump, and release mid-air.

> **⚠️ Warning:** The bicycle kick is enabled by default. If you find yourself accidentally flinging the ball backward while using weapons like Nagi's, you should disable this in the settings menu until you develop the necessary muscle memory.

### Weapon Stacking Mechanics
The developers have confirmed that **Weapon Stacking** is an intentional mechanic. This allows players to combine the best traits of two different weapons. For example, stacking **Emperor** with **Riptide** creates a shot that has the instant charge time of Emperor but retains the curve and power of Riptide.

Common stacking combinations include:
1.  **Emperor + Power Shot:** Deletes the long charge time of Power Shot while keeping the massive velocity.
2.  **Emperor + Direct Shot:** Allows for instant, high-powered volleys.
3.  **Formless + Dragon Header:** Used by elite players to gain massive height before delivering a powerful header.

## FAQ

**Q: How do I become an admin in a locked 2 private server?**

A: The person who purchases or creates the private server is the admin by default. They can grant admin status to others through the "Player" tab in the M command menu.

**Q: Can I use weapon stacking in public matches?**

A: Yes, weapon stacking is a core gameplay mechanic and is available in all game modes, including public matches and private server scrims. It is not considered an exploit.

**Q: What is the best ball gravity for a realistic match?**

A: The default ball gravity is **7**. Most competitive leagues stick to this value, though some experiment with 6.5 for a slightly "floatier" feel that mimics high-level football.

**Q: How do I change the map in my private server?**

A: You must first purchase the map in the in-game store using Yen. Once owned, go to your inventory, select the "Maps" tab, and equip the desired pitch before joining your server.
