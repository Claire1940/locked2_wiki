## Mengakses Menu Server Privat

Sebelum Anda dapat mulai memasukkan perintah, Anda perlu tahu di mana menemukan antarmukanya. Tidak seperti beberapa game yang murni mengandalkan perintah chat berbasis teks, judul ini menggunakan sistem berbasis UI yang tangguh. Untuk mengakses rangkaian perintah, Anda harus menjadi pemilik server privat atau telah diberikan izin admin oleh pemiliknya.

Setelah Anda berada di dalam server, navigasikan ke **Main Menu** (Menu Utama) dan cari tab **Commands** (Perintah). Ini adalah pusat untuk semua tindakan administratif. Jika Anda ingin bermain dengan orang-orang tertentu sebelum mendalami perintah, pastikan wilayah matchmaking Anda diatur dengan benar di bawah tab "Other" (Lainnya). Anda dapat memilih dari Amerika Utara, Amerika Selatan, Eropa, Asia, dan Oseania untuk memastikan latensi serendah mungkin bagi peserta Anda.

> **💡 Tip:** Jika Anda merasa terganggu dengan spam undangan party saat mencoba menyiapkan server, Anda dapat mematikan notifikasi undangan di menu pengaturan "Other" untuk menjaga fokus Anda.

## Perintah Locked 2 Penting untuk Pemilik Server

Inti dari manajemen server terletak pada toggle "Match" dan "Server". Ini memungkinkan Anda untuk menetapkan aturan dasar untuk sesi Anda. Misalnya, perintah **Server Lock** adalah garis pertahanan pertama Anda terhadap pemain yang tidak diinginkan yang bergabung di tengah pertandingan.

| Perintah | Kategori | Deskripsi |
| :--- | :--- | :--- |
| **Server Lock** | Server | Mencegah pemain baru untuk bergabung ke dalam instansi. |
| **Boss Spawning** | Gameplay | Mengaktifkan/menonaktifkan kemampuan boss untuk muncul dalam sesi. |
| **Cool downs** | Gameplay | Menghapus atau mengaktifkan cooldown kemampuan untuk latihan. |
| **Reset Ball** | Match | Secara instan memindahkan bola kembali ke tengah lapangan. |
| **Give Flow** | Player | Memberikan status "Flow" kepada pemain tertentu untuk pengujian atau latihan. |

Salah satu pertanyaan yang paling sering diajukan mengenai **perintah locked 2** adalah bagaimana sebenarnya cara memasukkan bola ke lapangan. Di PC, keybind default untuk memunculkan bola adalah **B**. Ini adalah perintah aksi cepat yang melewati menu, memungkinkan reset cepat selama latihan.

## Manajemen Pertandingan dan Alat Papan Skor

Menjalankan pertandingan yang terorganisir membutuhkan kontrol presisi atas waktu dan skor. Bagian "Match" dari menu perintah menyediakan beberapa alat yang bertindak sebagai wasit virtual. Anda dapat memulai atau menghentikan waktu permainan, memicu efek suara peluit untuk pelanggaran atau bola keluar lapangan, dan bahkan memaksa kickoff.

Papan skor juga sepenuhnya interaktif. Jika gol tercipta selama pemanasan dan Anda ingin menghapusnya sebelum pertandingan resmi dimulai, Anda dapat menggunakan fungsi **Reset Scoreboard**. Selain itu, Anda dapat mengganti visibilitas papan skor secara manual agar layar tetap bersih untuk rekaman sinematik.

### Kustomisasi Lapangan
Ukuran lapangan memainkan peran besar dalam meta permainan. Lapangan yang lebih kecil mendukung permainan intensitas tinggi dengan operan pendek, sementara lapangan besar lebih baik untuk menguji tembakan jarak jauh dan manajemen stamina. Melalui menu perintah, Anda dapat beralih antara versi lapangan **Small** (Kecil) dan **Large** (Besar) secara instan.

> **⚠️ Peringatan:** Mengubah ukuran lapangan di tengah pertandingan terkadang dapat menyebabkan pemain atau bola mengalami glitch dan menembus lantai. Sebaiknya atur ukuran lapangan sebelum pemain muncul atau selama reset pertandingan penuh.

## Manajemen dan Administrasi Pemain

Saat menyelenggarakan acara publik atau semi-privat, Anda mungkin bertemu dengan pemain yang tidak mengikuti aturan. Bagian manajemen pemain dari menu **perintah locked 2** memberi Anda beberapa cara untuk menangani peserta individu. Dengan mengklik nama pemain dalam daftar, Anda dapat mengakses sub-menu dengan opsi berikut:

| Tindakan | Hasil |
| :--- | :--- |
| **Admin** | Memberikan akses ke menu perintah kepada pemain tersebut. |
| **Referee Mode** | Menempatkan pemain dalam status netral untuk mengamati dan memimpin pertandingan. |
| **Kick** | Mengeluarkan pemain dari sesi saat ini. |
| **Blacklist** | Melarang pemain untuk bergabung kembali ke server. |
| **Teleport** | Memindahkan pemain ke lokasi Anda saat ini. |
| **Freeze** | Mengunci pemain di tempat, mencegah pergerakan. |

Alat-alat ini sangat penting untuk menjaga ketertiban. Misalnya, jika Anda sedang melakukan tutorial, "Membekukan" (Freeze) siswa memastikan semua orang mendengarkan dan tidak berlarian di sekitar lapangan.

## Pengubah Lanjutan dan Bot AI

Bagi pemain yang ingin melatih manuver ofensif mereka tanpa memerlukan tim penuh berisi sepuluh orang, perintah **AI Bot** adalah pengubah permainan. Anda dapat memunculkan AI Goalkeeper (GK) untuk menjaga gawang saat Anda melatih sudut tembakan Anda.

Untuk menempatkan AI GK, ikuti langkah-langkah berikut:
1. Berjalan ke posisi di mana Anda ingin bot berdiri (biasanya di tengah garis gawang).
2. Buka **Main Menu** dan navigasikan ke **Bots**.
3. Pilih **Place** (Tempatkan).
4. AI akan muncul dan segera mulai mempertahankan gawang.

Lebih jauh lagi, Anda dapat bereksperimen dengan "Multipliers" (Pengali). Dalam dunia **perintah locked 2**, multiplier bertindak sebagai pengubah yang dapat mengubah fisika, kecepatan, atau tingkat kekuatan. Ini sangat cocok untuk membuat mode permainan "mutator" yang memberikan istirahat dari rutinitas kompetitif standar.

## Pembuatan dan Kustomisasi Tim

Locked 2 memungkinkan kustomisasi berbasis tim yang mendalam langsung melalui antarmuka perintah. Anda tidak terbatas pada warna atau nama default. Dengan mengklik ikon "+" di menu Pemilihan Tim, Anda dapat membuat nama tim khusus (misalnya, "Priority" atau "Elite Strikers").

Kustomisasi juga meluas ke pakaian. Game ini mengambil pakaian avatar Roblox Anda saat ini (tanpa aksesori besar) untuk membuat kostum Anda. Namun, jika Anda ingin membuat tampilan khusus untuk tim Anda, Anda dapat menggunakan aksesori rambut dan masker tertentu yang diizinkan oleh game untuk tetap ada, memberikan identitas unik bagi skuad Anda di lapangan.

## Fitur Sinematik dan Sosial

Bagi pembuat konten, kamera **Free Roam** mungkin adalah alat yang paling kuat dalam persenjataan. Dengan menekan **Ctrl + Shift + P**, Anda dapat melepaskan kamera dari karakter Anda. Ini memungkinkan Anda untuk terbang di sekitar stadion, menangkap sudut sinematik dari gol, atau menonton pertandingan dari pinggir lapangan seperti penonton.

Fitur ini sering digunakan bersamaan dengan "Referee Mode" untuk memberikan tampilan gaya siaran pada pertandingan kompetitif. Ini adalah fitur wajib bagi siapa saja yang ingin merekam cuplikan berkualitas tinggi untuk media sosial atau sorotan turnamen.

## FAQ

**T: Bagaimana cara memunculkan bola di server privat?**

J: Jika Anda menggunakan PC, cukup tekan tombol **B**. Alternatifnya, Anda dapat menavigasi ke menu Commands dan memilih opsi "Reset Ball" untuk membawanya ke tengah lapangan.

**T: Bisakah saya memberikan akses ke perintah locked 2 kepada pemain lain?**

J: Ya. Dalam daftar pemain di menu perintah, pilih nama pemain dan klik tombol "Admin". Ini akan memungkinkan mereka menggunakan alat yang sama dengan pemilik server.

**T: Apa manfaat dari bot AI GK?**

J: AI GK adalah alat yang sangat baik untuk latihan solo. Ini memungkinkan penyerang untuk melatih penyelesaian akhir dan penggunaan "Flow" tanpa memerlukan pemain lain untuk menjaga gawang.

**T: Bagaimana cara mengubah wilayah matchmaking?**

J: Buka tab "Other" di Menu Utama dan klik "Matchmaking Region". Anda dapat memilih antara Amerika Utara, Amerika Selatan, Eropa, Asia, dan Oseania untuk mengoptimalkan koneksi Anda.
