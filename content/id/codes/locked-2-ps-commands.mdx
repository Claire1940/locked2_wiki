## Memulai: Matchmaking dan Pengaturan Sosial

Sebelum mendalami **perintah ps locked 2** yang spesifik, Anda harus memastikan server Anda dikonfigurasi untuk audiens yang tepat. Latensi tinggi dapat merusak pertandingan sepak bola, membuat pengaturan waktu dan aktivasi "Flow" hampir mustahil dilakukan. 

### Memilih Wilayah Anda
Langkah pertama bagi setiap host adalah menyetel wilayah matchmaking. Ini memastikan bahwa semua pemain yang terhubung ke sesi Anda mengalami ping serendah mungkin. Untuk mengubahnya, navigasikan ke tab "Other" di menu utama dan pilih "Matchmaking Region." Di tahun 2026, wilayah yang tersedia meliputi:
*   Amerika Utara (NA)
*   Amerika Selatan (SA)
*   Eropa (EU)
*   Asia
*   Oseania

### Mengelola Party Anda
Jika Anda sedang mempersiapkan sesi berbasis tim, gunakan daftar sosial di sisi kanan layar. Anda dapat melihat status pemain, termasuk apakah mereka sedang solo, dalam duo, atau bagian dari grup yang lebih besar (hingga empat pemain). Untuk mengundang teman, cukup klik ikon "Plus" di sebelah nama mereka. Jika Anda merasa kewalahan dengan permintaan, Anda dapat menonaktifkan "Party Invites" di menu pengaturan untuk menjaga privasi saat mengonfigurasi server Anda.

## Perintah PS Locked 2 yang Penting

Inti dari manajemen server privat terletak pada tab "Commands" di dalam menu utama. Antarmuka ini memungkinkan Anda untuk beralih pengaturan server global yang memengaruhi setiap pemain di lapangan.

| Perintah | Fungsi | Pengaturan Direkomendasikan |
| :--- | :--- | :--- |
| **Server Lock** | Mencegah pemain baru bergabung ke dalam sesi. | **On** untuk scrim privat. |
| **Boss Spawning** | Mengaktifkan atau menonaktifkan kemampuan untuk memunculkan entitas tingkat bos. | **Off** untuk permainan kompetitif. |
| **Cooldowns** | Mengaktifkan/menonaktifkan timer cooldown skill dan kemampuan. | **Off** untuk latihan intensif. |
| **Reset Ball** | Secara instan memindahkan bola kembali ke titik tengah. | Gunakan sesuai kebutuhan. |
| **Give Flow** | Memberikan status "Flow" kepada pemain tertentu atau diri sendiri. | Bagus untuk menguji ultimate. |

> **💡 Tips:** Jika Anda menggunakan PC, cara tercepat untuk memunculkan bola adalah dengan menekan tombol **B**. Pintasan ini adalah standar universal untuk "Ball" dan menghindarkan Anda dari navigasi menu di tengah latihan.

## Manajemen Pertandingan dan Kontrol Papan Skor

Bagi mereka yang menyelenggarakan turnamen atau pertandingan terorganisir, bagian Match dari **perintah ps locked 2** adalah perangkat utama Anda. Bagian ini mengontrol aliran waktu dan pelacakan poin.

### Kontrol Jam dan Peluit
Jam permainan sepenuhnya manual di server privat. Anda memiliki opsi untuk "Play" atau "Pause" timer kapan saja. Ini sangat berguna untuk sesi pelatihan di mana Anda mungkin ingin menghentikan aksi untuk menjelaskan kesalahan posisi taktis. Selain itu, perintah "Whistle" (Peluit) dapat digunakan untuk menandai pelanggaran atau akhir babak, memberikan pengalaman wasit yang imersif.

### Kustomisasi Lapangan
Tidak setiap pertandingan membutuhkan stadion besar. Locked 2 memungkinkan Anda untuk beralih antara ukuran lapangan "Kecil" (Small) dan "Besar" (Large). Lapangan kecil sangat bagus untuk pertandingan gaya "Street" 2v2 atau 3v3, sementara lapangan Besar adalah standar untuk permainan profesional 11v11.

| Fitur | Deskripsi | Kasus Penggunaan Terbaik |
| :--- | :--- | :--- |
| **Scoreboard** | Menampilkan gol saat ini untuk kedua tim. | Pertandingan Kompetitif |
| **Reset Scoreboard** | Menghapus semua gol dan meriset pertandingan ke 0-0. | Awal Pertandingan Baru |
| **Kickoff** | Meriset posisi pemain dan bola untuk awal yang baru. | Setelah gol tercipta |
| **Multipliers** | Berfungsi sebagai modifikasi gameplay (kecepatan, lompatan, dll.). | Mode Permainan Seru/Kustom |

## Admin dan Moderasi Pemain

Sebagai pemilik server, Anda memiliki otoritas "Admin" atas semua orang di instance tersebut. Ini sangat penting untuk mengeluarkan pemain yang mengganggu atau membantu seseorang yang mengalami glitch menembus peta.

Tindakan berikut dapat diterapkan pada pemain mana pun yang saat ini ada dalam daftar server:
1.  **Admin:** Memberikan akses menu perintah kepada pemain lain.
2.  **Referee Mode:** Menempatkan pemain dalam status netral untuk mengawasi pertandingan.
3.  **Kick:** Mengeluarkan pemain dari sesi saat ini.
4.  **Blacklist:** Melarang pemain secara permanen untuk bergabung kembali ke server privat Anda.
5.  **Teleport:** Memindahkan pemain ke koordinat Anda saat ini (berguna untuk mengumpulkan tim).
6.  **Freeze:** Membekukan pemain di tempat, mencegah pergerakan atau penggunaan kemampuan.

> **Peringatan:** Gunakan perintah Blacklist dengan bijak. Dalam komunitas Locked 2, memasukkan pemain ke daftar hitam tanpa alasan dapat mengakibatkan reputasi buruk di liga kompetitif.

## Fitur Lanjutan: Bot AI dan Pembuatan Tim

Salah satu pembaruan paling mengesankan di tahun 2026 adalah penyertaan Kiper (GK) AI. Jika Anda sedang melatih tembakan tetapi tidak memiliki teman untuk menjadi kiper, Anda dapat memunculkan asisten AI.

### Memunculkan Kiper AI
Untuk mengaturnya, berjalanlah ke area gawang tempat Anda ingin bot berdiri. Buka menu utama, navigasikan ke "Bots", dan pilih "Place". AI GK akan muncul secara instan. Bot ini sangat mumpuni dan akan bereaksi terhadap tembakan berdasarkan pengali server saat ini. Jika Anda ditugaskan ke sebuah tim, AI akan mengenali Anda sebagai sekutu dan bahkan dapat mengoper bola kembali kepada Anda setelah melakukan penyelamatan.

### Kustomisasi Tim
Menu "Team Selection" memungkinkan Anda membuat identitas khusus untuk pertandingan Anda. Dengan mengklik ikon "Plus", Anda dapat mengetikkan nama tim yang unik (misalnya, "Team Priority"). 

**Pakaian dan Topeng Kustom:**
Locked 2 menggunakan pakaian avatar Roblox Anda secara default, meskipun menghapus sebagian besar aksesori besar untuk menjaga performa. Namun, pemain menemukan bahwa aksesori rambut dan item wajah tertentu dapat digunakan untuk membuat "topeng" kustom atau tampilan tim yang unik. Jika Anda ingin tim Anda terlihat seragam, sarankan item katalog tertentu untuk dipakai rekan tim sebelum bergabung ke server.

## Kontrol Kamera dan Free Roam

Bagi pembuat konten atau analis, "Free Cam" adalah alat paling ampuh dalam gudang senjata **perintah ps locked 2**. Dengan menekan **Ctrl + Shift + P**, Anda masuk ke mode kamera bebas.

Ini memungkinkan Anda untuk:
*   Merekam cuplikan gol yang sinematik.
*   Mengamati formasi tim dari sudut pandang burung (bird's-eye view).
*   Memantau seluruh lapangan tanpa harus hadir secara fisik sebagai karakter pemain.

Fitur ini sangat diperlukan untuk sesi "VOD Review" di mana seorang pelatih menganalisis pergerakan penyerang dan bek mereka untuk meningkatkan performa di pertandingan liga berikutnya.

## FAQ

**Q: Bagaimana cara memunculkan bola menggunakan perintah ps locked 2 di ponsel?**

A: Meskipun pemain PC dapat menggunakan tombol "B", pemain seluler dan konsol harus membuka Menu Utama, navigasikan ke tab "Commands", dan klik tombol "Spawn Ball" atau "Reset Ball" secara manual. Saat ini tidak ada gestur sentuh khusus untuk memunculkan bola dalam build 2026.

**Q: Bisakah saya menyimpan pengaturan tim di server privat?**

A: Saat ini, nama tim dan konfigurasi khusus akan di-reset setelah instansi server privat ditutup. Direkomendasikan untuk menyimpan catatan berisi pengaturan dan nama tim pilihan Anda untuk dimasukkan kembali dengan cepat saat memulai sesi baru.

**Q: Apa yang sebenarnya diubah oleh "Multipliers"?**

A: Multiplier berfungsi sebagai modifikasi gameplay. Mereka dapat meningkatkan kecepatan bola, gravitasi dunia, atau kekuatan tendangan karakter Anda. Kebanyakan server privat kompetitif menjaga pengaturan ini di angka 1.0 untuk mencerminkan pengalaman Ranked resmi.

**Q: Apakah ada cara untuk mencegah orang menggunakan flow di server saya?**

A: Ya, Anda dapat menonaktifkan Flow secara efektif dengan menyalakan "Cooldowns" dan menyetel pengali perolehan Flow ke nol jika versi server tersebut mendukungnya. Alternatifnya, Anda dapat menggunakan "Referee Mode" untuk menghapus Flow secara manual dari pemain yang menggunakannya selama sesi latihan terbatas.
